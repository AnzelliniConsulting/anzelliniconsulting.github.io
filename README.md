# Anzellini Archaeological Consulting — Website Template
## Files
- `index.html` — Main single-page website
- `style.css`  — All styles

## Deploy
Upload both files to GitHub Pages, Netlify, or any static host.
All images are inline SVGs — no external image files needed.

## Contact Form
To activate the contact form without a server warning, connect EmailJS:
1. Sign up at emailjs.com
2. Replace YOUR_PUBLIC_KEY / YOUR_SERVICE_ID / YOUR_TEMPLATE_ID in the script block at the bottom of index.html
